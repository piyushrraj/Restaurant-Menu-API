# 🍽️ MenuManager Pro

A full-stack Restaurant Menu Management System built using Flask and MongoDB.  
This application allows users to efficiently manage menu items with complete CRUD functionality through a clean and modern UI.

---

## 🚀 Features

- ➕ Add new menu items  
- 📋 View all menu items  
- ✏️ Edit existing items  
- ❌ Delete items with confirmation  
- 🎨 Clean and responsive UI  
- ☁️ MongoDB Atlas integration  
- 🔐 Secure database connection (TLS)

---

## 🛠️ Tech Stack

- Backend: Python, Flask  
- Database: MongoDB Atlas  
- Frontend: HTML, CSS  
- Styling: Custom CSS (modern UI design)
